Complete guide to host your django-website on heroku

1. Copy your project separately.

2. Go to the official website of heroku and create your account.

3. The Heroku CLI requires git, so download git from it's official website.

By default, the 'git cmd' and 'git bash' will be installed along with git.

4. Go to the folder in which you have copied your files.

5. Open the command prompt or terminal there and navigate to that folder.

6. Install python virtual environment using the command -

pip install pipenv

7. You can check the git version using the command -

git --version

8. Login to heroku from the Command Line using the command -

heroku login

Please make sure your directory structure is as follows :

Folder_name

  • django_project

9. Create virtualenv using command -

virtualenv .

10. Activate the virtualenv using command -

.\Scripts\activate

11. Now your directory structure should be like :

Folder_name

  • django_project
  • Lib
  • Scripts
  • pyvenv.cfg

12. In the virtualenv, install django using the command in the terminal -

pip install django

13. Go to the project directory (where manage.py file exists) and create the requirements.txt file using the command - 

pip freeze > requirements.txt

14. Go to the root directory of your project and create a file 'Procfile'.

Note : Be sure the file 'Procfile' doesn't carry any extension.

15. Edit the 'Procfile' and insert the code below :

web: gunicorn your_project_name.wsgi
Note : Be sure the name of your project is correct otherwise it is not gonna be properly installed in heroku server.

16. Install gunicorn using the command -

pip install gunicorn

17. Install django-heroku using the command -

pip install django-heroku

18. Import django-heroku at the top of the settings.py file of your django-project.

import django_heroku

19. Add the settings at the bottom of the settings.py file of your django-project.

django_heroku.settings(locals())

20. Add the extra modules to your requirements.txt file.

pip freeze > requirements.txt

21. Create your heroku app using the command -

heroku create your_web_name
Note : Be sure the name of your website is unique.
Check if your website has been created successfully.

22. Go to the root directory of your project and initialize git using the command -

git init
Check if a git repository has been created successfully.

23. You can check your git status using the command -

git status

24. Add all the files using the command -

git add --all

25. Commit something to git using the command -

git commit -m "your_commit"
eg. git commit -m "First commit"

26. Run the command -

heroku git:remote -a your_web_name

27. Push all the files using the command -

git push heroku master

28. Everything is done now..... Your app is live...

29. You can check it on -

your_web_name.herokuapp.com

Updating your website

In case you have make changes in your website, in order to update these changes, follow the following steps :

1. Go to the project folder and open check the git status in the command prompt using the command -

git status

2. If there are any changes, then add these changes using the command -

git add --all

3. Commit these changes using the command -

git commit -m "commit_message"

4. Push these changes to heroku server using the command -

git push heroku master

5. And it's done..... Your changes are updated.
