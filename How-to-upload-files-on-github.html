Uploading files on github

1. Go to the github official website and create your account.

2. Then create your repository.

3. Go to the folder in which you have stored your files.

4. Open the command prompt or terminal there and navigate to that folder.

5. Initialise a git repository using the command -

git init

6. Add the file using the command -

git add file_name

or

7. Add all files using the command -

git add .

or

git add --all

or

git add -A

8. Commit the text using the command -

git commit -m "commit_name"

9. Copy the origin url of your repository.

10. Run the command -

git remote add origin origin_url

11. Push your file to your git repository using the command -

git push -f origin master

All done, Enjoy...
